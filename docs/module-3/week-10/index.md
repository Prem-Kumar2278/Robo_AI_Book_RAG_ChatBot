# Module 3 - Week 10: Navigation Pipeline

## Learning Objectives

By the end of this week, you will be able to:
- Implement navigation pipelines using Isaac ROS
- Configure path planning and obstacle avoidance
- Integrate localization and mapping systems
- Optimize navigation performance for various environments

This chapter is divided into several lessons that cover different aspects of navigation pipelines.
