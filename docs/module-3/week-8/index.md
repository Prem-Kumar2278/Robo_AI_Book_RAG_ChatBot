# Module 3 - Week 8: Isaac ROS Fundamentals

## Learning Objectives

By the end of this week, you will be able to:
- Understand the fundamentals of NVIDIA Isaac ROS
- Set up Isaac ROS development environments
- Implement basic perception and navigation components
- Integrate Isaac ROS with existing systems

This chapter is divided into several lessons that cover different aspects of Isaac ROS fundamentals.
