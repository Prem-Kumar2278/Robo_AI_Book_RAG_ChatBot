# Module 4 - Week 13: Capstone Project

## Learning Objectives

By the end of this week, you will be able to:
- Apply all learned concepts in a comprehensive project
- Integrate multiple systems and technologies
- Demonstrate advanced robotics capabilities
- Present and document project outcomes effectively

This chapter is divided into several lessons that cover different aspects of the capstone project.
