# Module 4 - Week 11: Vision-Language-Action Models

## Learning Objectives

By the end of this week, you will be able to:
- Understand Vision-Language-Action (VLA) models
- Implement VLA systems for robotic control
- Integrate multimodal AI models with robotics
- Apply VLA models to real-world robotic tasks

This chapter is divided into several lessons that cover different aspects of Vision-Language-Action models.
