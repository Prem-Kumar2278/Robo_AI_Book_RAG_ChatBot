# Module 1 - Week 3: Services and Actions

## Learning Objectives

By the end of this week, you will be able to:
- Implement services for request/response communication
- Use actions for goal-oriented communication
- Understand the differences between services and actions
- Apply advanced communication patterns in ROS 2

This chapter is divided into several lessons that cover different aspects of ROS 2 services and actions.
